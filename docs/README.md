# Documentation Status

The operational release source of truth is `product_release/`.

Files in this `docs/` directory are historical product and architecture references. They may lag current launch decisions, release gates, billing/trial behavior, STT mode caveats, and deployment evidence. For launch or soft-release decisions, use:

- `product_release/RELEASE_DECISION.md`
- `product_release/TESTER_RELEASE_MATRIX.md`
- `product_release/SOFT_RELEASE_TESTER_INSTRUCTIONS.md`
- `product_release/PUBLIC_LAUNCH_LEDGER.md`
- `product_release/PRECEDENCE.md`
